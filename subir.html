📋 Instrucciones

  1. Selecciona la categoría del producto
  2. Escribe el nombre y el precio
  3. Selecciona la imagen del producto
  4. Copia el código generado
  5. Ve a GitHub → repo Kihsu3d/Kihsu3d → carpeta content/[categoría]/
  6. Crea un archivo nuevo: nombre como quieras + extensión .md (ej: naruto.md)
  7. Pega el código copiado dentro del archivo
  8. Haz Commit changes
  9. La imagen se muestra automáticamente en la web en 1-2 minutos

💡 Para la imagen: súbela primero a GitHub en la carpeta images/ o usa un servicio como imgbb.com para obtener una URL directa.
